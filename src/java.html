Introducción a Java

Java es un lenguaje de programación de propósito general, concurrente, orientado a objetos y basado en clases. Fue diseñado para tener la menor cantidad posible de dependencias de implementación. Esto permite que los desarrolladores escriban el código una vez y lo ejecuten en cualquier lugar (WORA), lo que significa que el código Java puede ejecutarse en todas las plataformas que soportan Java sin necesidad de recompilación.
